Verilog Core Labs
Digital Logic Design using Verilog HDL

A comprehensive collection of Verilog HDL implementations covering Combinational Logic, Sequential Logic, and Finite State Machines (FSMs). This repository serves as a learning resource for students, FPGA enthusiasts, and VLSI beginners to understand RTL design, simulation, and digital circuit implementation.
The repository contains synthesizable Verilog modules along with testbenches and simulation files for fundamental digital design concepts.

Project Objectives
Learn Verilog HDL from basic to intermediate level.
Understand different Verilog modeling styles.
Design and verify combinational logic circuits.
Implement sequential circuits using flip-flops and registers.
Design Mealy and Moore Finite State Machines.
Develop reusable RTL modules.
Perform functional simulation and waveform analysis.
Build a strong foundation for FPGA and ASIC design.
