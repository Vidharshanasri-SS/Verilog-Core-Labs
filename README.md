Verilog-Core-Labs
This repo documents my hands-on journey into the world of VLSI and FPGA design -starting from absolute zero. From writing my first Verilog module to understanding how logic maps onto real silicon, everything I learn lives here. Built with Xilinx Vivado, tested on real FPGA hardware, and grounded in the fundamentals that chip designers actually use.

Unlike standard academic exercises, these modules are written with an "Hardware-First" mindset, ensuring they are ready for FPGA synthesis.

This process have helped me gain knowledge about the fundamentals of the 
=> Verilog
=> Timing concepts, Clock Domain Discipline, Reset Strategies, Timing Hazards Control
=> LUT consumption, power and area consumption
=> Hardware minimization
=> Combinational Logic & Optimization, Structural vs. Behavioral Modeling, Data Bus Manipulation
=> Sequential Logic & Synchronization
=> Testbench, Testbench Infrastructure, Directed Testing, Waveform Diagnostics
